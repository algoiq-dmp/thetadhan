🔮 Heikin Ashi AI — Color-Change Alert System

AI-powered trade signal generation from Heikin Ashi chart color transitions. Identifies trend reversals with automated stop-loss computation and confidence scoring.

Signal Detection Algorithm

SIGNAL: Color Change Detection (5-min chart) When 5-min HA flips from RED → GREEN: → Generate BUY alert → SL = min(low of last 3 red candles) → Confidence = 50 + (consecutive_greens × 10) + (no_lower_wick ? 15 : 0) → "STRONG BUY" if 3+ consecutive green with no lower wick When 5-min HA flips from GREEN → RED: → Generate SELL alert → SL = max(high of last 3 green candles) → Confidence = 50 + (consecutive_reds × 10) + (no_upper_wick ? 15 : 0) → "STRONG SELL" if 3+ consecutive red with no upper wick
